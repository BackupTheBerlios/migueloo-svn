Para crear un curso debe de completar los siguientes campos que son obligatorios:

  • Titulo: Este campo identificará a la asignatura.
  • Facultad: Aquella en la que pertenece la asignatura. Esta se seleccionará de entre las que aparecen en el menú desplegable.
  • Código del curso: Código asignado para la identificación del curso. Si en su organización no le asignan un código de curso, usted se lo puede inventar (recuerde que es &uuacte;nico y no puede tener el mismo que otros cursos).
  • Titular: Aquí se hará constar el que será el titular o encargado de coordinar el curso seleccionado.
  • Idioma: En esta última casilla haremos constar el idioma que se utilizará a lo largo del curso.

Una vez rellenados todos los campos se pulsará sobre el botón 'Validar' apareciendo una nueva página en la que se nos indica que se ha creado la web del curso con el código que introdujimos en la página anterior. Esta página nos aparece un vínculo que nos llevará directamente al curso que hemos creado.

Esto es un generador de preguntas. Permite la realización de un examen de respuesta múltiple. De todas formas, son formativos, ya que el sistema no es lo suficientemente seguro para asegurar el correcto recuento de las preguntas del test.

El principio de este sistema de puntuación es definir el valor del numero de respuestas correctas por pregunta. Por ejemplo, si tienes una pregunta con 4 respuestas (posiblemente mas de una correcta) y tiene dos mal y dos bien, puede tener la mitad de los puntos, pero también puede ser que el profesor vea que no es lo correcto y darle, por ejemplo, la máxima puntuación (20) si todo es correcto y cero puntos para cualquier otra combinación.
